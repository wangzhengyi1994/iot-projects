# 物联网项目集合

IoT 项目前端静态页面仓库, 纯 HTML + CSS + JS, 部署在 GitHub Pages

## 项目列表

| 项目 | 目录 | 说明 |
|------|------|------|
| 一感智能科技 (YIGAN) | `/yigan-website/` | 智慧水务一站式交付平台官网 |
| 奥德帕斯 (ODPS) | `/odps-website/` | 物联网综合管理平台官网 |

## 技术栈

- HTML5 + CSS3 + JavaScript (ES6+)
- 无框架依赖, 纯静态页面
- 字体: OPPOSans, DingTalk JinBuTi, Noto Sans SC
- 设计规范: Arco Design 色阶体系
- CSS 变量驱动的主题系统

## 部署

GitHub Pages 自动部署, 访问对应项目的 index.html 即可
